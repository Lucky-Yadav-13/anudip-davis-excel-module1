# Amazon Sales Data Analysis (Excel Project)

## Overview
This project analyzes **Amazon sales data using Microsoft Excel** to extract useful business insights from the dataset.  
The analysis focuses on understanding order trends, product performance, shipping details, and revenue distribution.

The dataset contains information about orders including order ID, order date, product category, quantity sold, revenue amount, fulfillment type, and shipping location.

Excel tools such as **Pivot Tables, Filters, and Sorting** were used to summarize and analyze the data.

---

## Dataset Information

The dataset includes the following fields:

| Column | Description |
|------|-------------|
| Order ID | Unique identifier for each order |
| Date | Date when the order was placed |
| Status | Status of the order |
| Fulfilment | Method used to fulfill the order |
| Sales Channel | Platform through which the order was placed |
| Ship Service Level | Shipping service used |
| Style | Product style |
| SKU | Stock keeping unit |
| Category | Product category |
| Size | Product size |
| ASIN | Amazon Standard Identification Number |
| Courier Status | Delivery status of the courier |
| Qty | Quantity ordered |
| Currency | Currency used in transaction |
| Amount | Total order value |
| Ship City | City where the order was delivered |
| Ship State | State where the order was delivered |
| Ship Postal Code | Postal code of delivery location |
| Ship Country | Country of delivery |
| B2B | Indicates whether the order is business-to-business |

## Project Structure

The Excel file contains multiple sheets:

| Sheet Name | Description |
|------------|-------------|
| amozon | Raw dataset containing all order records |
| Pivot Tables | Sheets containing pivot table analysis |

---

## Analysis Performed

The following analysis was conducted using **Pivot Tables**:

- Sales distribution by **product category**
- Revenue analysis by **state and city**
- Order count based on **order status**
- Quantity analysis based on **product size**
- Sales comparison based on **fulfillment methods**
- B2B vs B2C order comparison
- Shipping and courier performance analysis
